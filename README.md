# mlops-from-scratch

MLOps from scratch project.

## Requirements

- Python >= 3.9

## Setup

### Install dependencies

```bash
make install
```

### Install development dependencies

```bash
make install-dev
```

## Development

### Format code

```bash
make format
```

### Run linting

```bash
make lint
```

### Run tests

```bash
make test
```

### Run all (format + lint + test)

```bash
make all
```

### Clean up

```bash
make clean
```
