# Project Instructions

## Commit Convention

**중요**: 사용자가 커밋을 요청하면 반드시 `/commit` 스킬을 사용하세요.

커밋 관련 요청 예시:
- "커밋해줘"
- "commit"
- "변경사항 커밋"
- "git commit"

이런 요청이 들어오면 자동으로 `/commit` 스킬을 실행합니다.

## Development Commands

```bash
make install-dev  # 개발 의존성 설치
make format       # 코드 포맷팅
make lint         # 린트 검사
make test         # 테스트 실행
```
